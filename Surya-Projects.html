PROJECTS

Monocular Visual Odometery using 5 Point Algorithm and LK Tracking

Developed a python-based module for observing visual monocular images to get the odometry of the robot. The Algorithm uses Feature point detection (SIFT, SURF, ORB) and LK Tracking of the feature to initialize a 5-point algorithm and recover pose to get a odometry data. The module was tested and benchmarked using KITTI-00 Dataset and was observed a 92% IOU.

Path planning for Autonomous Drone Delivery

Developed a ROS based node for path planning and delivering of a box to a particular coordinate. Used Bug-0 based path planning and obstacle avoidance was achieved by using a top mounted 4 direction radar. The Autonomous drone delivery was simulated in the gazebo platform. The box that has to be picked up had a QR code attached to it. The drone has to scan and get the delivery coordinate. Pyzbar package was used for this purpose. And the delivery location is dynamically initialized in the simulation.

ROS Based Autonomous Drone simulation

Developed a ROSpy based control system for a quadcopter to transverse to a set of GPS setpoint autonomously. The Control System has two modules namely Altitude controller and the position controller, Altitude controller stabilize the drone at the zero error Roll, Yaw, Pitch angles using a PID based controller, the position controller takes in the target GPS coordinate has setpoint values and calculates the roll yaw pitch angles to successfully move to the setpoint coordinates. these controllers work in synchronisation to autonomously fly the drone from one coordinate to another

Visual SLAM for Autonomous Ground Robot

Implemented RTABMAP-ROS visual SLAM on a ground robot using a RGBD camera to map a 3D environment. Real time appearance-based mapping (RTAB-Map) was used to make the 3D map launched in gazebo environment. 

SLAM for Autonomous Ground Robot

Implemented Gmapping using 2D lidar for turtlebot in ROS to construct a 2D map in an indoor environment and used adaptive Monte Carlo based navigation stack to move the ground robot to a navigation goal

Autonomous Line Follower Robot

Advanced PID based line follower equipped with the ability to find the shortest path using BUG2 algorithm alongside the PID algorithm. The controller's PID algorithm restores the measured speed to the desired speed with minimal delay and overshoot. Bug2 algorithm employs greedy algorithm and it will take the first path that looks better.
Algorithm: PID and BUG2 algorithm.
Competitions Won: 17 (from 01/2019 - 03/2020)
Components used: N20 Micro Geared Motors, MDD10A, Arduino Nano, 16 sensor array, 3D printed Chassis[PLA] & Silica wheels.

Omega-Mars VTOL

VTOL was designed and engineering design report was presented for the international planetary aerial vehicle competition as a part of team ATOM. Omega is capable for locomotion in Mars and the Software Architecture makes it completely autonomous. It uses ROS based system architecture and add on tools like Orthomosaic capture, Atmosphere analysis unit. SLAM and autonomous unit makes it a signature UAV for Martian use.
